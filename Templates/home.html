About This Project


This site demonstrates the use of machine learning to predict employee job satisfaction. The system uses a MongoDB database to store user inputs and Python to process the data. Users can fill out a simple form and choose between two prediction models: Random Forest or Linear Regression. Once submitted, the system runs the data through the selected model and generates results that highlight predicted levels of employee satisfaction.


Future development will explore integrating the prediction system into an AI chatbot.
